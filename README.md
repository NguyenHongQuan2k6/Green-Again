#  Project Overview

**SSG: Green Again** is an educational open-world Role-Playing Game (RPG) developed in **Roblox Studio**.

The project combines environmental education with engaging gameplay mechanics, allowing players to restore polluted ecosystems, recycle waste, rescue wildlife, and rebuild nature while learning sustainable environmental practices.

Instead of simply presenting environmental issues, the game encourages players to actively participate in solving them through interactive missions and rewarding progression systems.

#  Gameplay

Players become environmental volunteers responsible for restoring polluted areas.

Throughout the journey, players will:

-  Collect scattered waste
-  Sort recyclable and non-recyclable materials
-  Plant trees to restore ecosystems
-  Rescue wildlife
-  Interact with educational NPCs
-  Unlock cleaner environments as progress increases

Every completed mission contributes to rebuilding a healthier ecosystem.

#  Educational Objectives

Green Again aims to raise awareness about environmental protection by teaching players how to:

- Reduce pollution
- Protect ecosystems
- Recycle responsibly
- Preserve biodiversity
- Participate in community-driven environmental restoration

The game transforms environmental education into an engaging interactive experience.
